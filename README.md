# 🏫 SALEM HILLS INT'L SCHOOL AI STUDIO

An advanced, expert educational assistant web application custom-built for Salem Hills International School. Powered by **Google Gemini 2.5 Flash**, this workspace features tailored pedagogical instruction parameters and dynamic document compilation capabilities.

---

## 🚀 Features

*   **Intelligent Chat Workspace:** Designed around robust educational frameworks. Automatically structures responses using step-by-step reasoning and precise lower-case Roman numerals ($i, ii, iii$) for clarity.
*   **Visual Processing Indicators:** Leverages active user-interface execution wrappers (`st.spinner`) to clearly communicate prompt processing and token generation cycles.
*   **Native Multi-Modal Utilities:** Built-in infrastructure configured for native Gemini text-to-image generation configurations.
*   **Sanitized PDF Compilations:** Generates an official, publication-ready PDF transcript of active chat sessions. The exporter automatically filters out conversational interface metadata and chatbot instructions to yield a highly professional document.
